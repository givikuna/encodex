# encodex

[![npm version](https://img.shields.io/npm/v/@givi-tsvariani/encodex.svg)](https://www.npmjs.com/package/@givi-tsvariani/encodex)

An ambitious answer to extending TypeScript's functionality in all possible ways. Very much still in beta.

## ⚠️ Under Construction (The Great Rework)
`encodex` is currently undergoing a complete rewrite from scratch to ensure maximum modularity, strict typing, and proper organization. We are building a massive, modular ecosystem ranging from standard data structures to quantum computing simulations.

## Installation

```bash
npm install @givi-tsvariani/encodex
```

## Modules Map
Once the rework is complete, encodex will export the following highly-modular libraries:

### 🛠 Core & Utilities
* **Utils**: Massive collection of utility functions for strings, integers, arrays, and functions (e.g., map, fold, chunk, memoize, debounce, clamp, lerp).
* **Types**: Useful types for extending TS functionality (e.g., Nullable).
* **Interfaces**: Standardized interfaces (`Cloneable, Filterable, Mappable, Collection, Observable, etc.).
* **Alternative**: Alternative types as classes replacing standard primitives (EDouble, EInteger, EString, EList).
* **Chronos**: Time calculation tools (IntervalAl, Epoch, Relativistics, Recurrence).

### 📚 Computer Science Fundamentals
* **Structures**: Comprehensive data structures (BST, LLRB Tree, Graphs, Heaps, Trie, SkipList, RingBuffer, etc.).
* **Sorting**: Every sorting algorithm you could need, from standard (QuickSort, MergeSort) to esoteric (BogoSort, StalinSort, SpaghettiSort).
* **Monads**: Functional programming constructs (Maybe, Either, Result, Task, State).
* **OOPTools**: Helpers for Object-Oriented design (Mixins, Observer, Singleton, Factory, EventEmitter).

### ➗ Mathematics & Logic
* **Math**: A massive module containing sub-modules for LinAl (Linear Algebra), AbsAl, CatTheory, NumTheory, Trig, Combinatorics, Pathfinding, Crypto, SetTheory, Prob (Probability & Stats), Chaos, Geometry, and more.
* **Lang**: Tools for creating programming languages (Token, Scanner, ASTNode, ScopeManager, RecursiveDescent).
* **Onto**: Knowledge representation (Triplestore, Inference, Taxonomy).

### 🌌 Physics & Quantum Computation
* **Physics**: Simulations and calculations across Kinematics, Orbital, Wave, RigidBody, Relativity, Astro, QM (Quantum Mechanics), Chem, Optics, and Fluids.
* **QC**: Quantum Computation simulation (Qubit, QuantumRegister, CustomGate, Circuit, Algo, QuantumNoise).

### 🎲 Game Development
* **Game**: Common game-dev systems including Dice (notation), Chance, Noise, LootTable, Inventory, Combat formulae, Grid generation, and generic Progression blueprints.
